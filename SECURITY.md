# Security Policy

## Supported Versions

Only the latest released version of jwt_inspector is supported with security
updates.

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly. Do not open a public issue.

Send an email to: **stepan.kutaj@tldr-it.com**

Please include:

- A description of the vulnerability.
- Steps to reproduce the issue.
- Any potential impact or severity assessment.

You can expect an initial acknowledgment within 72 hours. We will work with you
to understand the issue and coordinate a fix before any public disclosure.
